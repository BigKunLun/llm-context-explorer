# React Context Explorer

一个用于可视化 LLM（大语言模型）Context 学习过程的 React 应用。

## 功能

- 可视化展示 LLM 的 Context 消息流
- 展示每步的 Token 使用情况
- 演示 Context 学习场景

## 技术栈

- React 19
- TypeScript
- Vite
- Tailwind CSS 4

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览构建结果
npm run preview
```
